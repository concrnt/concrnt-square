"Concrnt"という名前は厳密には「分散型SNS**エンジン**」の名前であり、アプリケーションの名前ではありません。Concrntで作られたアプリケーションとして [concrnt.world](https://concrnt.world)があります。

ここでアカウントを作成することで、Concrntを活用した分散型SNSを始めることができます！


使い方などのチュートリアルは、使い始めた後にconcrnt.world内から参照することができます。
